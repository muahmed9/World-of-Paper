# نظام عالم ورق الكاشير - مولد الفواتير الذكي

تطبيق ويب احترافي لتوليد وإدارة قوائم البيع وفواتير "عالم ورق الكاشير" في بغداد.

## كيفية النشر على GitHub Pages

هذا المشروع مهيأ للعمل تلقائياً باستخدام **GitHub Actions**. لضمان عمل الموقع بشكل صحيح، اتبع الخطوات التالية:

1. ارفع جميع ملفات المشروع إلى مستودع (Repository) جديد على GitHub.
2. اذهب إلى إعدادات المستودع (**Settings**).
3. اختر **Pages** من القائمة الجانبية.
4. تحت قسم **Build and deployment**، قم بتغيير المصدر (**Source**) إلى **GitHub Actions**.
5. انتظر دقيقة واحدة، وسيكون موقعك جاهزاً للعمل على الرابط الذي سيوفره لك GitHub.
